# Habit Tracker

Habit Tracker is a React Native-based mobile application designed to help users build and maintain good habits. With this app, users can set goals, track their progress, and stay motivated to achieve their daily or long-term habits.

## Features

- Create and manage habits.
- Track daily progress.
- View habit statistics and insights.
- User authentication for personalized habit tracking.

## Getting Started

Follow these steps to set up and run the project locally:

### Prerequisites

- Node.js installed on your system.
- Expo CLI installed globally. You can install it using:

  ```bash
  npm install -g expo-cli
  ```

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd habit-tracker
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

### Running the App

1. Start the development server:

   ```bash
   npx expo start
   ```

2. Choose one of the following options to open the app:

   - Use a development build.
   - Open it in an Android emulator or iOS simulator.
   - Scan the QR code with the Expo Go app on your physical device.

### Resetting the Project

If you want to reset the project to a blank state, run:

```bash
npm run reset-project
```
